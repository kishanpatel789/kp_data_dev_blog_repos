Positional and Keyword
Arguments

Common Patterns

              f(x1, /, x2, *, x3)   # pos-only, pos-or-kw, kw-only
            
              f(x1, /, *, x2)       # positional-only, keyword-only
            
              f(x1, x2, /)          # all params positional only
            
              f(*, x1, x2)          # all params keyword only

When to use each parameter type?

  1. Position-only: Order of arguments matters
  2. Keyword-only: Improve readability when the function is called
  3. Position-or-Keyword: It doesn't matter
