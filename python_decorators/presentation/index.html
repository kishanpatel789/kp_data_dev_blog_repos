Decorator Syntax


@my_decorator

def my_func():
    print("Hello world")

my_func = my_decorator(my_func)
            

@my_decorator
def my_func():
    print("Hello world")

Use Decorators to...

  • Register functions with a registry
  • Measure time performance of functions
  • Restrict function calls to users with permissions
  • Emit logs each time functions are called
  • Filter the inputs and outputs of functions
  • Cache the output of a function for reuse
  • Limit how often functions can be called
  • Pay your taxes

import time

def rate_limit(func):
    last_called = 0
    def wrapper(*args, **kwargs):
        nonlocal last_called
        now = time.time()
        if now - last_called <= 10:
            raise Exception("Rate limit exceeded. Wait!")
        last_called = now
        return func(*args, **kwargs)
    return wrapper

A decorator is...

  • a function ...
  • that takes another function ...
  • and returns a function ...
  • potentially extending it along the way.

A decorator is...

  • a callable ...
  • that takes another callable ...
  • and returns a callable ...
  • potentially extending it along the way.
