Common Encoding Patterns

  1. Plain Encoding: When there's just nothing better.
  2. Dictionary Encoding: When there are a few unique values that appear a lot.
  3. Run Length Encoding: When there are runs of repeated values.
  4. Delta Encoding: When there are large numbers that are sorted and increasing slowly.
